# covid19-data-analysis
This project is a Power BI dashboard that visualizes COVID-19 data across different countries. 
It shows Confirmed Cases, Deaths, Recovered, and Active Cases and provides insights into the most affected countries.
- Horizontal Bar Charts for each metric: Confirmed, Deaths, Recovered, Active
- Interactive visualization with clear country labels
- Insights based on data trends
- Overall, the USA has the highest number of confirmed cases, deaths, and active cases among all countries.
- India and Pakistan show significant recovered cases.
- Active cases are highest in the USA, followed by India and Pakistan.
- This project is created for learning and practicing Power BI data visualization techniques with real-world COVID-19 data.
